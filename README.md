# docker-compose-laravel
This is an example of a simple docker compose configuration to run a Laravel project.

### Run Laravel at: http://localhost:8000/
### Run Phpmyadmin site at: http://localhost:8080/

## More details here

| Server     | Port |
|------------|------|
| MySQL      | 8989 |
| PHPMyAdmin | 8080 |
| Nginx      | 8000 |
| Nginx SSL  | 3000 |
